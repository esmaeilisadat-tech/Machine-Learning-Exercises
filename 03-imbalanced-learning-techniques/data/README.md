# Datasets for this Module

## Credit Card Fraud Data
**Filename:** `creditcard.csv`
**Description:** Highly imbalanced dataset of credit card transactions for fraud detection.
**Source:** [Download Link](https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud)

> Note: Only a small sample (`sample_creditcard.csv`) is provided in this repository to prevent large file size issues. Please use the source link to download the full dataset for execution.
