# Datasets for this Module

## Energy Data
**Filename:** `energy_data.csv`
**Description:** Dataset containing features related to energy consumption.
**Source:** [Download Link](https://archive.ics.uci.edu/ml/datasets/Appliances+energy+prediction)

> Note: Only a small sample (`sample_energy_data.csv`) is provided in this repository to prevent large file size issues. Please use the source link to download the full dataset for execution.

## Life Expectancy Data
**Filename:** `life_data.csv`
**Description:** Dataset for predicting life expectancy based on health and economic factors.
**Source:** [Download Link](https://www.kaggle.com/datasets/kumarajarshi/life-expectancy-who)

> Note: Only a small sample (`sample_life_data.csv`) is provided in this repository to prevent large file size issues. Please use the source link to download the full dataset for execution.
