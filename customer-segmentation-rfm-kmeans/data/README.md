# Datasets for this Module

## Mall Customers Data
**Filename:** `mall_customers.csv`
**Description:** Dataset for customer segmentation using K-Means.
**Source:** [Download Link](https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python)

> Note: Only a small sample (`sample_mall_customers.csv`) is provided in this repository to prevent large file size issues. Please use the source link to download the full dataset for execution.
