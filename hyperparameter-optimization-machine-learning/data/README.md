# Datasets for this Module

## Mobile Price Data
**Filename:** `mobile_price_data_train.csv`
**Description:** Dataset for classifying mobile phone prices.
**Source:** [Download Link](https://www.kaggle.com/datasets/iabhishekofficial/mobile-price-classification)

> Note: Only a small sample (`sample_mobile_price_data_train.csv`) is provided in this repository to prevent large file size issues. Please use the source link to download the full dataset for execution.
