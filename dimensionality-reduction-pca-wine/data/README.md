# Datasets for this Module

## Wine Quality Data
**Filename:** `winequality-red.csv`
**Description:** Dataset containing physicochemical properties of red wine and quality ratings.
**Source:** [Download Link](https://archive.ics.uci.edu/ml/datasets/wine+quality)

> Note: Only a small sample (`sample_winequality-red.csv`) is provided in this repository to prevent large file size issues. Please use the source link to download the full dataset for execution.
